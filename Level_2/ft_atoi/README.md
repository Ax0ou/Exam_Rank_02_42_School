## Sujet

```BASH
Nom de l'exercice : ft_atoi
Fichiers attendus : ft_atoi.c
Fonctions autorisées : None
--------------------------------------------------------------------------------

Écrire une fonction qui converts the string argument str to an integer (type int)
and retourne it.

It works much like the standard atoi(const char *str) function, see the man.

Votre fonction doit être déclarée comme suit :

int	ft_atoi(const char *str);
```
