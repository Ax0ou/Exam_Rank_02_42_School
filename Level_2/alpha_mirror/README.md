## Sujet

```BASH
Nom de l'exercice : alpha_mirror
Fichiers attendus : alpha_mirror.c
Fonctions autorisées : write
--------------------------------------------------------------------------------

Écrire un programme nommé alpha_mirror that prend une chaîne and affiche this string
after replacing each alphabetical character by the opposite alphabetical
character, suivi d'un saut de ligne.

'a' becomes 'z', 'Z' becomes 'A'
'd' becomes 'w', 'M' becomes 'N'

and so on.

Case is not changed.

Si le nombre d'arguments n'est pas 1, display only a newline.

Exemples :

$>./alpha_mirror "abc"
zyx
$>./alpha_mirror "My horse is Amazing." | cat -e
Nb slihv rh Znzarmt.$
$>./alpha_mirror | cat -e
$
$>
```
