## Sujet

```BASH
Nom de l'exercice : reverse_bits
Fichiers attendus : reverse_bits.c
Fonctions autorisées :
--------------------------------------------------------------------------------

Écrire une fonction qui prend un octet, reverses it, bit by bit (like the
example) and retourne the result.

Votre fonction doit être déclarée comme suit :

unsigned char	reverse_bits(unsigned char octet);

Exemple :

  1 byte
_____________
 0010  0110
	 ||
	 \/
 0110  0100
```
