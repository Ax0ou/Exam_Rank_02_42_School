## Sujet

```BASH
Nom de l'exercice : swap_bits
Fichiers attendus : swap_bits.c
Fonctions autorisées :
--------------------------------------------------------------------------------

Écrire une fonction qui prend un octet, swaps its halves (like the example) and
retourne the result.

Votre fonction doit être déclarée comme suit :

unsigned char	swap_bits(unsigned char octet);

Exemple :

  1 byte
_____________
 0100 | 0001
     \ /
     / \
 0001 | 0100
```
