## Sujet

```BASH
Nom de l'exercice : is_power_of_2
Fichiers attendus : is_power_of_2.c
Fonctions autorisées : None
--------------------------------------------------------------------------------

Écrire une fonction qui determines if a given number is a power of 2.

This function retourne 1 if the given number is a power of 2, otherwise it retourne 0.

Votre fonction doit être déclarée comme suit :

int	    is_power_of_2(unsigned int n);
```
