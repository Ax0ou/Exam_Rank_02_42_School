## Sujet

```BASH
Nom de l'exercice : rev_print
Fichiers attendus : rev_print.c
Fonctions autorisées : write
--------------------------------------------------------------------------------

Écrire un programme qui prend une chaîne, and affiche the string in reverse
suivi d'un saut de ligne.

Si le nombre de paramètres n'est pas 1, le programme affiche a newline.

Exemples :

$> ./rev_print "zaz" | cat -e
zaz$
$> ./rev_print "dub0 a POIL" | cat -e
LIOP a 0bud$
$> ./rev_print | cat -e
$

```
